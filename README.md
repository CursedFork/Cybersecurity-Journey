# Cyber Certification Journey

This repository tracks my progress studying for IT and cybersecurity certifications, including:

- CompTIA A+ (Core 1 & 2)
- CompTIA Security+
- Pentesting labs and exercises

The goal of this journal is to document what I learn, reinforce understanding, and build a semi-professional portfolio of technical growth. 

---

## How to Use This Repository

- `week-XX.md` contains weekly journal entries: what I studied, practice scores, struggles, reflections, and next steps.
- `resources.md` lists study materials, guides, and practice test sites.
- `labs/` contains optional screenshots, notes, or scripts from hands-on labs.
