# Legacy Data Transfer from Windows XP

## Project Overview:
Successfully migrated files from a 2005 Dell PC running Windows XP to a modern system, overcoming USB compatibility challenges.

**Key Steps:**
1. Diagnosed USB detection issues on legacy hardware.
2. Prepared an 8GB FAT32 flash drive for cross-compatibility.
3. Transferred critical files while ensuring data integrity.

**Skills Demonstrated:**
- Legacy system troubleshooting
- File system and USB compatibility management
- Data migration and verification
